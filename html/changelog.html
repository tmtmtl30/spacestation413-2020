ATHATH updated:

ArcaneMusic updated:

  • The Medical NanoMed vending machine has been split into two, with the all new NanoDrug vending machine taking most chemical medicines.
  • Literally everything purchasable in a vending machine has had it's prices adjusted to better suit players to buy more things, while still enabling them to interact with each other using money.
  • Player passive income has been removed.
  • Departmental budgets income has been thoroughly reworked, as they no longer need to balance the whole of the crew's income off of them, making them a more reasonable resource pool scaled to the station's size or efforts.
  • Departmental income is a static value for every department, getting more funds every 5 minutes (Plus whatever's added from vending machines!)
  • Buying goods through your department's vendors now provides an 80% discount as opposed to being completely free.
  • Butchering mobs with embedded objects inside of them will now drop the embedded objects as well.
